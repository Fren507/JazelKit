## English Version

JazelKit is a simple Vanilla JavaScript/TypeScript framework for web projects. It provides a basic structure and tools to create interactive, modular web applications that run directly in the browser.

### Features

- Simple and easy-to-use API
- Support for real-time features via WebSockets
- Modular design for easy extension and customization
- Hot-reload support for faster development
- Example applications to get you started
- Written in TypeScript for type safety and modern JavaScript features
- Lightweight and fast
- Open source and free to use

### Getting Started

Clone the repository and install dependencies:

```bash
git clone https://github.com/Fren507/JazelKit.git
cd JazelKit
npm install
```

To run the demo application:

```bash
npm run demo
```

Then open `http://localhost:5173` in your web browser.

Alternatively, you can delete the `demo` folder and use the `main` folder as your project base. To run the main application, run:

```bash
npm run dev
```

This starts a local server to access the example applications in your browser.

## Deutsche Version

JazelKit ist ein einfaches Vanilla JavaScript/TypeScript Framework für Webprojekte. Es bietet eine grundlegende Struktur und Werkzeuge, um modulare, interaktive Webanwendungen direkt im Browser zu erstellen.

### Funktionen

- Einfache und benutzerfreundliche API
- Unterstützung für Echtzeit-Funktionen via WebSockets
- Modulares Design für einfache Erweiterung und Anpassung
- Hot-Reload-Unterstützung für schnellere Entwicklung
- Beispielanwendungen zum Einstieg
- Geschrieben in TypeScript für Typsicherheit und moderne JavaScript-Funktionen
- Leichtgewichtig und schnell
- Open Source und kostenlos nutzbar

### Erste Schritte

Repository klonen und Abhängigkeiten installieren:

```bash
git clone https://github.com/Fren507/JazelKit.git
cd JazelKit
npm install
```

Demo-Anwendung starten:

```bash
npm run demo
```

Dann `http://localhost:5173` im Webbrowser öffnen.

Alternativ kann der `demo`-Ordner gelöscht und der `main`-Ordner als Projektbasis genutzt werden. Zum Starten der Hauptanwendung:

```bash
npm run dev
```

Dies startet einen lokalen Server, um die Beispielanwendungen im Browser zu nutzen.
